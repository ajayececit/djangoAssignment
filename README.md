APIs:
Add users : http://localhost:8000/addusers/
Remove Users : http://localhost:8000/removeuser/{id}
Show users : http://localhost:8000/api/user/
sort by last name : http://localhost:8000/api/sortlastname/
sort by first name : http://localhost:8000/api/sortfirstname/
